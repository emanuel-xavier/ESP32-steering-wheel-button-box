ButtonBox Config

ESP32 Steering Wheel — Wi-Fi Configuration

1
Enter config mode Hold Button 1 (pin 2) while powering on the device.
2
Connect to the Wi-Fi network Look for ButtonBox-Config in your Wi-Fi list and connect to it.
3
Open the config page Navigate to http://192.168.4.1 in any browser.
4
Save your settings Edit the values and press Save & Reboot. The device reboots as a gamepad.
Open Config Page →
The config page is served directly by the ESP32 — only reachable while connected to ButtonBox-Config Wi-Fi.
