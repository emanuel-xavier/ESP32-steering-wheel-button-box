ESP32 Button Box

BLE gamepad for a steering wheel button box — configurable at runtime via Wi-Fi, no reflashing needed.

ESP32 BLE Gamepad 14 Buttons 2 Encoders Wi-Fi Config

How to configure

1
Enter config mode Hold Button 1 (pin 2) while powering on the device.
2
Connect to the Wi-Fi network Look for ButtonBox-Config in your Wi-Fi list and connect to it.
3
Open the config page Navigate to http://192.168.4.1 in any browser — the page is served by the ESP32 itself.
4
Save & reboot Edit any setting and press Save & Reboot. Settings persist in NVS across power cycles.

Features

14 Physical Buttons Hardware debounced via Bounce2, fully mapped to BLE gamepad buttons.
2 Rotary Encoders Microsecond hardware debounce. Each rotation fires a timed gamepad button press.
Zones Encoder Mode One encoder selects a zone; the other triggers different buttons per zone — multiplying inputs without extra hardware.
Persistent Config All settings stored in ESP32 NVS. Survives power loss. Changed via Wi-Fi, no USB needed.
BLE HID Gamepad Works on Windows, macOS, Linux, Android and iOS as a standard gamepad — no drivers.
Serial Debug Enable #define SERIAL_DEBUG to log all button and encoder events at 115200 baud.
